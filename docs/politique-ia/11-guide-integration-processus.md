# Guide d’intégration IA dans les processus

## Objectif
Intégrer l’IA sans casser le process métier.

## Étapes
1. Identifier le point de gain
2. Définir rôle IA (assistante)
3. Valider outputs humains
4. Mesurer gain

## Exemple
- Processus : relance devis
- IA : rédaction brouillon
- Validation : commercial
