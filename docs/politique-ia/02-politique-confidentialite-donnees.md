# Politique de confidentialité & données sensibles (IA)

## Objectif
Protéger les données confidentielles utilisées avec les outils d’IA.

## Classification des données
- **Public** : partage autorisé
- **Interne** : usage interne uniquement
- **Confidentiel** : accès restreint
- **Sensible** : interdiction d’IA externe

## Règles
- Aucune donnée **Sensible** dans un outil d’IA externe.
- Données **Confidentielles** uniquement avec anonymisation.
- Archivage des échanges IA selon la politique IT.

## Anonymisation minimale
- Retirer noms, N° contrat, montants exacts, adresses, identifiants.

## Responsable
DPO / Responsable conformité.
