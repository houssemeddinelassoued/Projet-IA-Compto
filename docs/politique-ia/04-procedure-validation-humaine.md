# Procédure de validation humaine

## Objectif
Assurer un contrôle systématique des outputs IA.

## Niveaux de validation
- **N1** : vérification simple (collaborateur)
- **N2** : validation manager (documents externes)
- **N3** : validation juridique / DG (contrats, finance, RH)

## Étapes
1. Relecture factuelle
2. Vérification des chiffres
3. Validation de conformité
4. Approbation finale

## Traçabilité
Conserver : prompt, version du document, validateur.
