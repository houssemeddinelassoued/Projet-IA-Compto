# Charte d’utilisation de l’IA

## Objectif
Définir les règles simples d’usage de l’IA au sein de Compto.

## Périmètre
Tous les collaborateurs, stagiaires, prestataires utilisant des outils d’IA.

## Règles essentielles
- Utiliser l’IA pour assister, jamais remplacer la responsabilité humaine.
- Ne pas partager de données sensibles ou confidentielles.
- Vérifier toutes les sorties avant diffusion.
- Citer l’aide de l’IA quand un livrable externe est produit.

## Données interdites
- Données clients sensibles (IBAN, CNSS, pièces d’identité)
- Données RH (salaires, dossiers personnels)
- Clauses contractuelles non anonymisées

## Bonnes pratiques
- Masquer ou anonymiser les fichiers
- Privilégier des prompts clairs et objectifs
- Conserver une trace du prompt si livrable officiel

## Validation
Toute utilisation à impact légal/commercial doit être validée par le responsable.
