# Modèle d’évaluation des risques IA

## Objectif
Mesurer les risques avant déploiement.

## Critères
- Impact métier
- Données sensibles
- Risque légal
- Risque réputationnel

## Échelle
Faible / Moyen / Élevé.

## Décision
- Faible : OK
- Moyen : validation manager
- Élevé : validation DG/Juridique
