# Guide de prompting par métier

## Objectif
Aider chaque métier à formuler des prompts efficaces.

## Comptabilité
- Exemple : analyser anomalies, doublons, dates aberrantes.

## Contrats / Juridique
- Exemple : extraire clauses clés + risques + tableau synthèse.

## Commercial
- Exemple : relance client, synthèse offre, points clés.

## Direction
- Exemple : rapport mensuel, synthèse KPIs, recommandations.

## Modèle de prompt
1. Rôle
2. Contexte
3. Tâches précises
4. Format de sortie
5. Contraintes
